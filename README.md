# Agentic Writer AI Bot

An AI-powered creative writing assistant built with Python.

## How to Install and Run

### 1. Install Dependencies
Run this command in your terminal to install the required libraries:

pip install google-genai matplotlib pandas anyio httpx google-auth

### 2. Set Your Gemini API Key
Make sure to set your Gemini API key as an environment variable before running the project.

### 3. Run the Project
1. Open the `.ipynb` notebook file in VS Code or Jupyter Notebook.
2. Run all cells sequentially.
3. Follow the interactive menu in the console to generate text and view visual analytics.

---

## Tech Stack

* Language: Python
* AI Framework: Google GenAI SDK
* Data & Analytics: Pandas, Matplotlib
